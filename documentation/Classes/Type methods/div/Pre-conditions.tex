\tt{super(((Object *) dividend)->type, ((Object *) divisor)->type)}.

denote the value of \tt{((Object *) result)->type}.

\begin{itemize}

\item \tt{is_type(}$type$\tt{)} must be \tt{true}.

\item \tt{is(}$type$\tt{,} $super$\tt{)} must be \tt{true}.

\end{itemize}

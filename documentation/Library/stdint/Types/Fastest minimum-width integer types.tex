\tt{UInt_fast}$N$\_ is a synonym for \tt{uint_fast}$N$\tt{_t}, and
\tt {Int_fast}$N$\_ is a synonym for  \tt{int_fast}$N$\tt{_t}.

\type{UInt_fast32} & \type{UInt_fast64}\\

\type {Int_fast32} & \type {Int_fast64}\\
